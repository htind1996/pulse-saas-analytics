# Pulse SaaS Analytics Platform

Pulse is an end-to-end SaaS product analytics project.

## Current Architecture

User  
↓  
HTML / CSS Frontend  
↓  
Flask Backend  
↓  
MySQL Database  

## Current Features

- Multi-step user registration
- Flask session handling
- Password hashing
- MySQL integration
- Parameterized SQL inserts
- User profile and business data collection

## Technology Stack

- HTML
- CSS
- Python
- Flask
- MySQL
- SQL

## Project Goal

The goal of this project is to build an end-to-end product analytics pipeline from application data generation to SQL-based analytical modelling and Power BI reporting.
